Instructions

install all ghost deps & ghost cli

curl -sL https://deb.nodesource.com/setup_12.x | sudo -E bash -
sudo apt-get install -y nodejs
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
brew install node
brew install wget
sudo apt-get install build-essential
npm install ghost-cli@latest -g

start ghost

mkdir ghost && cd ghost
ghost install local --no-start --enable --port 2368

go to https://alexkleindevs.github.io in browser

Create an account, read the intial blog post how to edit with Markdown etc.

ghost start

make static export directory and generate static site from your running ghost blog

mkdir website && gssg --dest website --url 'https://yourusername.github.io'

upload to Github Pages - GUI

Make a repository with yourname.github.io as the project name, drag contents of website (or whatever your gssg output was) to github repo root folder

(for WSL only to find files in GUI)
